# App React + Vite (lista para correr)

## Requisitos
- Node.js 18+
- npm

## Pasos
1. Instalar dependencias:
   ```bash
   npm install
   ```
2. Levantar en desarrollo:
   ```bash
   npm run dev
   ```
3. Build de producción:
   ```bash
   npm run build
   npm run preview
   ```

## Firebase
Asegurate de completar `src/firebaseConfig.js` con tus credenciales válidas.
